# WoodMaster - Sistema de Gestão para Marcenarias (SaaS)

![WoodMaster Logo](https://raw.githubusercontent.com/alecsanderlima97/woodmaster/main/public/logo.png) (Placeholder para logo)

O **WoodMaster** é uma plataforma SaaS premium desenvolvida pela **Orqeustracs**, projetada especificamente para simplificar e otimizar a gestão de marcenarias modernas. Com foco em design impecável e funcionalidade robusta, o sistema oferece uma experiência completa desde a captura de leads até a entrega final dos projetos.

## 🚀 Funcionalidades

- **Dashboard Inteligente**: métricas em tempo real sobre vendas, projetos e faturamento.
- **CRM Integrado**: Funil de vendas, gestão de clientes e histórico de interações.
- **Gestão de Projetos**: Controle de prazos, orçamentos e etapas de fabricação.
- **Módulo Financeiro**: Fluxo de caixa, controle de pagamentos e faturamento.
- **Design Premium**: Interface moderna, responsiva e otimizada para produtividade.

## 🛠️ Tecnologias

- **Frontend**: [Next.js](https://nextjs.org/) (App Router)
- **Estilização**: [Tailwind CSS](https://tailwindcss.com/)
- **Backend**: [Firebase](https://firebase.google.com/) (Auth, Firestore, Hosting)
- **Animações**: [Framer Motion](https://www.framer.com/motion/)
- **Ícones**: [Lucide React](https://lucide.dev/)

## 📦 Instalação e Execução

1. Clone o repositório:
   ```bash
   git clone https://github.com/alecsanderlima97/woodmaster.git
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

3. Configure o Firebase no arquivo `.env.local`:
   ```env
   NEXT_PUBLIC_FIREBASE_API_KEY=seu_api_key
   NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=seu_auth_domain
   NEXT_PUBLIC_FIREBASE_PROJECT_ID=seu_project_id
   NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=seu_storage_bucket
   NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=seu_sender_id
   NEXT_PUBLIC_FIREBASE_APP_ID=seu_app_id
   ```

4. Execute o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
